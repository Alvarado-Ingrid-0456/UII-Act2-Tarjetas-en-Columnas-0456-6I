# PROMPT
Desarrolla una pantalla en Flutter para la aplicación Papelería Cometa, mostrando una sección de productos titulada "ÚTILES", basada en un diseño de tarjetas en columna similar al ejemplo proporcionado.
🎨 Estilo visual:
Mantener la identidad de Papelería Cometa:
Azul intenso (#0D47A1 aprox.) como color principal
Morado/violeta como color secundario
Fondo gris claro con tono lila suave
Diseño moderno, limpio y juvenil
Bordes redondeados en todos los contenedores
Sombras suaves para efecto de elevación
Evitar el uso de .withOpacity()

🔵 AppBar:
Fondo azul fuerte
Íconos en blanco
Título centrado con:
Logo dentro de un contenedor blanco con bordes redondeados
Texto: "Papelería Cometa" en blanco

🟣 Body:
📌 Título:
Mostrar el texto "ÚTILES" centrado (NO en el AppBar)
Color morado
Tamaño grande, en mayúsculas

📦 Tarjetas (IMPORTANTE):
Crear 5 tarjetas en columna
Cada tarjeta debe seguir la estructura del ejemplo:
Estructura de cada tarjeta:
Contenedor con:
Fondo blanco
Bordes redondeados
Sombra suave
Distribución horizontal (Row):
Lado izquierdo:
Imagen del producto (desde GitHub)
Tamaño aproximado: 80x80 a 100x100
Lado derecho:
Nombre del producto (texto grande)
Categoría o descripción corta (texto secundario)

Cada tarjeta debe tener efecto hover (para web/desktop):
Aumentar ligeramente la elevación
Cambiar sutilmente el borde a color morado
Escalar ligeramente (zoom leve)

📐 Detalles adicionales:
Espaciado uniforme entre tarjetas
Márgenes amplios
Todo alineado correctamente
Mantener consistencia visual con toda la app
Diseño limpio, sin saturación
